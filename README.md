## English

### Overview

Baneronetwo Web is a modern web application built with React, TypeScript, and Tailwind CSS. It features a responsive design with a rich set of UI components powered by Radix UI and styled with Tailwind CSS.

### Features

- **Modern UI Components**: Accordion, dropdown menus, navigation menus, select inputs, and more
- **Responsive Design**: Fully responsive layout that works on all devices
- **Type Safety**: Built with TypeScript for better developer experience and code quality
- **Fast Development**: Powered by Vite for lightning-fast builds and hot module replacement

### Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

### Tech Stack

- **Frontend Framework**: React
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Build Tool**: Vite
- **UI Components**: Radix UI primitives
- **Form Handling**: React Hook Form
- **Data Visualization**: Recharts
- **Routing**: React Router
